# Setup

* Assumes `.bashrc` contains:

```shell
if [ -f ~/.bash_extras ]; then
    . ~/.bash_extras
fi
```

* Configures:
  * [autojump](https://github.com/wting/autojump)
  * Bash prompt
  * [fzf](https://github.com/junegunn/fzf)
